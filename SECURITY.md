# Güvenlik Politikası

## Açık Kaynak & Hassas Veriler

Bu repoda **şifre, API anahtarı veya token bulunmaz.**

- `.dev.vars` — gitignore'da, commit edilmez
- Tüm hassas değerler `env` / `wrangler secret` üzerinden okunur
- `.dev.vars.example` yalnızca şablon (boş placeholder), gerçek değer içermez

## Güvenlik Açığı Bildirimi

Bir güvenlik açığı keşfettiğinizde lütfen **public issue açmayın.** Bunun yerine:

1. E-posta ile bildirin: [Bilge Karga iletişim]
2. Mümkünse repro adımları ve etki analizi ekleyin
3. Cevap süresi: 48 saat içinde ilk yanıt

## Sorumlu Açıklama

- Kritik açıklar için 90 gün koordinasyon süresi
- Düzeltme sonrası public disclosure (mutabık kalınırsa)
